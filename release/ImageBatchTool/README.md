# ImageBatchTool - 专业图片批处理工具

## 📦 版本信息
- **版本号**: v1.0.0
- **发布日期**: 2026-02-06
- **编译方式**: MSVC 动态链接（商业级发布）

## ✨ 功能特点
✓ **Photoshop 级自由变换**（Ctrl+T）  
✓ **批量处理图片**（调整大小、裁剪、对齐）  
✓ **支持格式**: JPG/PNG/BMP/TGA  
✓ **可视化画布系统**  
✓ **高质量输出**（JPG 95质量 / PNG 无损）  

## 🚀 快速开始
1. 解压文件到任意目录
2. 双击运行 `ImageBatchTool.exe`
3. 无需安装任何依赖，开箱即用

## ⌨️ 快捷键

### 变换模式（Ctrl+T）
- `Ctrl+T` - 进入/退出变换模式
- `Enter` - 确认变换
- `ESC` - 取消变换
- 拖拽边缘中点 - Photoshop级等比例缩放（固定对边）
- 拖拽角点 - 等比例缩放（固定对角）
- 拖拽图片 - 移动图片
- 方向键 - 微调图片位置（1px）
- `Shift+方向键` - 快速移动（10px）

### 画布视图
- `Alt+滚轮` - 缩放画布视图（以鼠标为中心）
- `空格+拖拽` - 平移画布（抓手工具）
- `Ctrl+滚轮` - 画布左右移动
- 鼠标滚轮 - 画布上下滚动
