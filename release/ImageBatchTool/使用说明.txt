========================================
  图片批处理工具 v1.0 - Photoshop级自由变换
========================================

【功能特点】
  ✓ Photoshop 工业级自由变换（Ctrl+T）
  ✓ 批量处理图片（调整大小、裁剪、对齐）
  ✓ 支持 JPG/PNG/BMP/TGA 格式
  ✓ 可视化画布系统
  ✓ 高质量输出（JPG 95质量 / PNG 无损）

【使用方法】
  1. 双击运行 ImageBatchTool.exe
  2. 点击"添加文件夹"或"添加图片"导入素材
  3. 在右侧设置画布尺寸，点击"应用画布"
  4. 按 Ctrl+T 进入变换模式，调整图片位置和大小
  5. 点击"开始处理"选择输出目录

【系统要求】
  ✓ Windows 10/11（64位）
  ✓ 无需安装任何依赖
  ✓ 绿色软件，解压即用

【快捷键 - 变换模式（Ctrl+T）】
  Ctrl+T       - 进入/退出变换模式
  Enter        - 确认变换
  ESC          - 取消变换
  拖拽边缘中点 - Photoshop级等比例缩放（固定对边）
  拖拽角点     - 等比例缩放（固定对角）
  拖拽图片     - 移动图片
  方向键       - 微调图片位置（1px）
  Shift+方向键 - 快速移动（10px）

【快捷键 - 画布视图】
  Alt+滚轮     - 缩放画布视图（以鼠标为中心）
  空格+拖拽    - 平移画布（抓手工具）
  Ctrl+滚轮    - 画布左右移动
  鼠标滚轮     - 画布上下滚动

【技术亮点】
  ✓ 完全按照 Photoshop 工业级标准实现
  ✓ 边缘约束缩放（数学上绝对不漂移）
  ✓ 智能对齐辅助线
  ✓ 双精度浮点计算（零累积误差）

【Photoshop 级自由变换说明】
  拖拽下边缘中点时：
  - 上边缘完全固定（数学上绝对不动）
  - 保持宽高比进行等比例缩放
  - 无任何边缘漂移或抖动
  - 符合 Photoshop 工业级标准
